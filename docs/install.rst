Installation
============

The software is distributed via PyPI_, it can be installed with ``pip``:

::

    pip install mutalyzer-crossmapper


From source
-----------

The source is hosted on GitHub_, to install the latest development version, use
the following commands.

::

    git clone https://github.com/mutalyzer/crossmapper.git
    cd crossmapper
    pip install .


.. _PyPI: https://pypi.org/project/mutalyzer-crossmapper
.. _GitHub: https://github.com/mutalyzer/crossmapper.git
